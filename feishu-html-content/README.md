# feishu-html-content

将生成的 HTML 文件通过飞书云文档分享给用户。

## 核心功能

将 HTML 代码完整复制到飞书云文档中，而非作为附件（附件方式无法达到预期效果）。

## 适用场景

- 分享 HTML 演示文稿
- 分享网页样式代码
- 分享任何 HTML 格式的内容

## 文件大小处理

| 场景 | 处理方式 |
|------|----------|
| 小文件 (< 10KB) | 直接完整复制到云文档 |
| 大文件 (>= 10KB) | 分块粘贴，每块 < 10KB |

## 分块策略

对于大文件，会自动拆分：

1. 将 HTML 拆分成若干部分（每块 < 10KB）
2. 优先在标签结束处 (`>`) 分割，保证 HTML 结构完整
3. 添加分段标记 `<!-- 第 X/Y 部分 -->`
4. 依次写入同一飞书云文档
5. 输出完整文档链接

## 输出格式

完成后返回飞书云文档链接，格式：
```
飞书云文档：https://open.feishu.cn/document/<doc_token>
```

## 相关技能

- **feishu-doc**: 飞书文档核心操作
- **frontend-slides**: HTML 演示文稿生成
