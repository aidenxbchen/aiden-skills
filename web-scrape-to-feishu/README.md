# web-scrape-to-feishu

> 从网页抓取结构化信息，汇总写入飞书多维表格

## 适用场景

- 浏览社交媒体（小红书、微博等）账号的热门内容，统计互动数据
- 从网页列表中批量提取信息，整理成飞书表格
- 任何「网页数据采集 → 结构化存储」的任务

## 依赖工具

| 工具 | 用途 |
|------|------|
| Safari MCP | 浏览网页、执行 JavaScript 提取数据 |
| Feishu/Lark MCP | 创建多维表格、写入记录 |

## 前置条件

1. Safari 已启用「允许 Apple 事件中的 JavaScript」
2. 飞书 MCP 已配置且具备 Bitable 读写权限

## 流程概览

```
网页列表页 ──JS提取──→ 缩略数据 ──排序筛选──→ Top N URL
     │
     ▼
详情页逐一访问 ──JS提取──→ 完整数据
     │
     ▼
飞书建表（无公式）──等待15s──→ 逐条写入记录 ──→ 返回表格链接
```

## 预计耗时

~5 分钟（抓取 10 条详情 + 建表 + 写入）

## 实战案例

**任务**：统计小红书 Garment Lab 最火的 10 篇笔记的点赞、收藏、评论数据

**结果**：5 分钟内完成数据抓取并生成飞书多维表格，包含排名、标题、点赞数、收藏数、评论数字段
