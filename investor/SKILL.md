# investor - 投资人会议文档命名管理技能

## 技能简介

帮助陈曦斌管理投资人会议文档，按照统一规则重命名文件并整理到指定文件夹。

## 触发词

- "投资人文档"
- "重命名会议文档"
- "整理投资人沟通文件"
- "investor"

## 使用前提

需要先开启飞书云盘工具权限（feishu_drive_file），否则无法访问云盘文件。

## 文件命名规则

### 规则格式

```
【机构名】- 投资人名字-日期.pdf/docx
```

### 示例

- 【启明创投】- 印慧欣-2026-03-13.pdf
- 【五源资本】- 杨煜珂-2026-03-22.docx
- 【华山资本】- 线下-2026-03-23.pdf

### 规则说明

1. 机构名使用【】包裹
2. 投资人名字紧跟机构名后面，用"-"分隔
3. 日期格式：YYYY-MM-DD
4. 如有线上/线下标注，加在日期前面

## 核心功能

### 1. 根据投资人名字查找会议时间

用户告知投资人名字后，助手会：
1. 搜索用户日历中与该投资人相关的会议
2. 提取会议时间和日期
3. 确认会议详情

### 2. 重命名文件

用户发送需要重命名的文件（通过消息发送文档）时：
1. 下载文件
2. 结合用户提供的投资人信息
3. 查找对应日程获取准确日期
4. 按照规则生成新文件名

### 3. 文件移动

将重命名后的文件移动到目标文件夹（投资人沟通/Angel round+）

## 使用流程

1. **用户提供信息**：
   - 投资人/机构名称
   - 原始会议文档

2. **助手查找日程**：
   - 在日历中搜索对应会议
   - 获取准确日期和时间

3. **执行操作**：
   - 重命名文件
   - 移动到指定文件夹
   - 删除原文件（如需要）
